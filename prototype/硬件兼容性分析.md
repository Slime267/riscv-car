## 核心变更

### 2WD vs 4WD

| | 4WD | **2WD** |
|------|------|------|
| 电机数 | 4 | **2** |
| 驱动方式 | 左右各 2 并 | **左右各 1** |
| 支撑 | 无 | **万向轮(含)** |
| 功耗 | ~800mA | **~500mA** |
| 速度 | — | 更快(载重轻) |
| L298N 单路电流 | 2×TT=800mA | **1×TT=400mA** ✅ |
| 控制逻辑 | 差速转向 | **差速转向(同)** |
| 价格 | ~¥15 | **~¥10** |
| 接线 | 复杂 | **简单** |

2WD 差速转向完全不受影响——左轮快右轮慢 = 右转，一正一反 = 原地旋转。

### L298N vs L9110S

| | L9110S | **L298N** |
|------|------|------|
| 单路电流 | 800mA | **2A** |
| 带 1 个 TT 电机 | 勉强 | ✅ 充裕 |
| 电压压降 | ~0.7V | **~2V** |
| 控制线/路 | 2 线 | **3 线** |
| 总引脚数 | 4 | **6** |
| 散热片 | 无 | 有 |

L298N 的 2V 压降是好事：7.4V 电池 → L298N 压降 → 电机实得 **~5.4V**，TT 电机额定 3–6V，5.4V 在安全范围内，无需 PWM 限幅。

## 供电架构

```
18650 × 2 (7.4V)
    │
    ├──→ MP1584 → 5V ──→ ESP32C3 5V 引脚 (主控供电)
    │                  └─→ L298N 5V 引脚 (逻辑供电, 去掉跳线帽!)
    │                  └─→ ESP32C3 3.3V → SSD1306 VCC
    │
    └──→ L298N 12V 引脚 (电机供电, 直连电池 7.4V)

全部 GND 共地
```

### ⚠️ L298N 跳线帽必须去掉

L298N 模块上有一个 5V 跳线帽（靠近 5V 排针）。**不拔掉会怎样**：模块用自带的 7805 从 7.4V 生成 5V 逻辑电。7.4V 对 7805 来说偏低（需要 ≥7.5V），电池电压一下降 7805 就失稳，导致 L298N 逻辑异常、电机乱转。

**操作**：拔掉跳线帽，把 MP1584 的 5V 接到 L298N 的 5V 排针上——逻辑电源由外部稳定 5V 提供。

## 引脚分配

```
合宙 ESP32C3 经典版 GPIO 占用:

  GPIO0  → ADC 电池电压 (10K+20K 分压后)
  GPIO1  → L298N IN4
  GPIO2  → L298N ENA   (左轮 PWM)
  GPIO3  → L298N IN1
  GPIO4  → L298N IN2
  GPIO5  → L298N ENB   (右轮 PWM)
  GPIO6  → SSD1306 SDA (I2C)
  GPIO7  → SSD1306 SCL (I2C)
  GPIO8  → 板载 LED
  GPIO10 → L298N IN3

  剩余空闲: GPIO9, GPIO20, GPIO21
```

## 功耗与续航

```
满负载:
  TT 电机 × 2 @ 5.4V:  2 × 400mA = 800mA
  ESP32C3 BLE:                     80mA
  OLED:                             5mA
  L298N 逻辑:                      10mA
  ─────────────────────────────────────
  总计:                          ~895mA

MP1584 3A → 仅用 100mA(逻辑+MCU) → 不热
L298N 2A/路 → 400mA 仅用到 20% → 温热
18650 2500mAh → ~900mA → 约 2.5 小时续航
```

## 综合判定

| 维度 | 评估 |
|------|------|
| **兼容性** | ✅ 全部匹配 |
| **电流余量** | ✅ L298N 2A 带 1×TT(400mA) 充裕 |
| **电压安全** | ✅ L298N 压降保护了电机 |
| **烧板风险** | ✅ 无，MP1584 隔离 |
| **外设数量** | ✅ 3 种 (BLE + PWM + I2C OLED) |
| **成本** | ✅ 比 4WD 方案更便宜 |

> **结论：可以采购。这个方案比之前 GA12-N20 + 4WD + L9110S 更合理——电机驱动余量大、接线直接、TT 电机速度快演示效果好。**
